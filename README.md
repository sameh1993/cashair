# 🛒 نظام الكاشير - POS System

## متطلبات التشغيل
- Node.js 18+
- MySQL 8+ أو MariaDB 10.5+

---

## خطوات التشغيل

### 1. إعداد قاعدة البيانات
```sql
-- شغّل الملف في MySQL Workbench أو phpMyAdmin
source database.sql
```

### 2. إعداد ملف البيئة
```bash
cp .env.example .env
```
ثم افتح `.env` وضع بيانات الاتصال بقاعدة البيانات.

### 3. تثبيت الحزم
```bash
npm install
```

### 4. تشغيل السيرفر
```bash
# للتشغيل العادي
npm start

# للتطوير (مع hot reload)
npm run dev
```

افتح المتصفح على: **http://localhost:3000**
